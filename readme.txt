== Installation ==

Install this plugin is very simple:

1. Upload 'login-logo-customization' folder to the `/wp-content/plugins/` directory
2. Activate  plugin through the 'Plugins' menu in WordPress
3. Edit "Dashboard--> Logo" settings
3. Place the Url of the Custom Logo
